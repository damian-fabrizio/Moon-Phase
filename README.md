# Moon-Phase

## This application displays the current state/phase of the moon alongside other relevant information using an API

## Function
Display the current date. Display the current phase that the moon is in an an accompanying image. Display when next full moon will be, when next new moon will be, the current age of the current moon, the date of the previous full moon, the illumination of the moon expressed as a percentage, and the distance in miles the moon is from Earth. In the background are cool stars that float on past the screen.

## Goals
Strengthen HTML, CSS, JavaScript and API use. Data is updated as often as the user makes a request. Intended to create pleasing UI that fit the tone of the application (night, stars, darkness) animated stars in background are a nice touch. 

## Fixes
Find ways to enhance UI. Potentially add more animations or other UI effects
